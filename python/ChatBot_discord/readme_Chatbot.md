## ChatBot_discord
> バージョン
Python 3.8.3
> 使用ライブラリ
discord.py, gspread
> ファイルの説明
- Sample_smash.py
"/tnam"コマンドを打ち込むと、チャンネル名に記載された大会番号に対応して結果をシートに書き込みます。
"/clist"コマンドを打ち込むと、ユーザーの固有IDを読み取って対応する列に記載してあるキャラクターを表示します。
コマンドに触れたことがない方でも使えるよう対話型のシステムを目指し、ユーザーが記入するまで待つ処理をつけました。
